# URL Shortener

A simple URL shortening web application designed to showcase different deployment approaches to cloud services. This project demonstrates how to deploy the web app using various infrastructure-as-code examples.

## Overview

This project comprises two main components:
1. **Web Application:** A Node.js and Express based URL shortening service that allows you to:
   - Generate unique short codes for given URLs.
   - Redirect from a short URL to the original URL.
   - Display a list of all shortened URLs.
2. **Infrastructure as Code:** Examples and templates for deploying the web app to cloud services. 

## Project Structure

- web
    - Contains the web application code.
- infrastructure
    - Contains infrastructure-as-code examples to deploy the web app.
    - Organized by cloud service provider and deployment approach.
